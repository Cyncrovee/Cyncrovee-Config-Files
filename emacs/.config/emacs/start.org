* Welcome to Emacs!

** The leader key is most likely set to space.

*** Quick Access
====== Open the Config Directory: "<leader> ffc" ======
====== Open a Project: "<leader> fps"            ======
====== Open the Buffer Menu: "<leader> bm"       ======
====== Exit Emacs: "<leader> eqa"                ======
